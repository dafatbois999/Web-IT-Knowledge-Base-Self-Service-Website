# IT Knowledge Base (ระบบคลังความรู้ไอที)

เว็บแอปพลิเคชันสำหรับรวบรวมและเผยแพร่วิธีแก้ไขปัญหาคอมพิวเตอร์เบื้องต้น (Knowledge Base) ช่วยให้ผู้ใช้งานสามารถค้นหาวิธีแก้ปัญหา Hardware, Software และ Network ได้ด้วยตนเอง พร้อมระบบหลังบ้าน (Admin Panel) สำหรับจัดการเนื้อหาครบวงจร

 **FEATURES**
###  (User Interface)
- **Live Search:** ค้นหาบทความแบบ Real-time (พิมพ์ปุ๊บ เจอปั๊บ) รองรับการค้นหาจากหัวข้อและเนื้อหา
- **Category Filter:** กรองบทความตามหมวดหมู่ (Hardware, Software, Network)
- **Responsive Design:** รองรับการใช้งานทั้งบนคอมพิวเตอร์และมือถือ
- **Sticky Navbar:** เมนูนำทางแบบติดขอบบน พร้อมช่องค้นหาที่ซ่อน/แสดงได้
- **Article Detail:** หน้าอ่านบทความที่สวยงาม รองรับการแสดงผลรูปภาพและวิดีโอ YouTube

### (Admin Panel)
- **Dashboard Overview:** สรุปยอดบทความทั้งหมดและแยกตามหมวดหมู่
- **Authentication:** ระบบ Login และเพิ่ม Admin ใหม่
- **CRUD Operations:** เพิ่ม, ลบ, แก้ไข และดูตัวอย่าง (Preview) บทความ
- **Rich Content Editor:** สามารถแทรกรูปภาพลงในเนื้อหา (Inline Image) ได้อย่างอิสระ
- **Image Upload:** ระบบอัปโหลดรูปภาพอัตโนมัติไปยัง Cloud Storage

## (Tech Stack)

- **Frontend:** HTML5, CSS3 (Bootstrap 5.3), JavaScript (ES6 Modules)
- **Backend / Database:** Supabase (PostgreSQL)
- **Storage:** Supabase Storage (สำหรับเก็บรูปภาพ)
- **Icons & Fonts:** Bootstrap Icons, Google Fonts (Sarabun)


**ตั้งค่า Supabase**
สร้าง Table ใน SQL Editor :

**Table: articles**
create table articles (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default timezone('utc'::text, now()),
  title text,
  category text,
  content text,
  solution text,
  image_url text,
  video_url text,
  status text default 'Published'
);

**Table: admins**
create table admins (
  id bigint generated by default as identity primary key,
  username text,
  password text
);

**Storage**: สร้าง Bucket ชื่อ images และตั้งค่า Policy ให้เป็น Public
